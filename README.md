# AdelPanel — Легковесная панель управления Linux-сервером

**AdelPanel** — это **open-source**, максимально простая и быстрая панель управления VPS/сервером на базе Linux.  
Она создана для разработчиков, фрилансеров и владельцев небольших проектов, которым нужно за считанные минуты превратить «голый» сервер в полноценный веб-хостинг — без терминала, без лишних функций и без огромного потребления ресурсов.

![Интерфейс AdelPanel — минималистичный дашборд](https://adelpanel.net/panel.jpg)  
*Главный экран AdelPanel: всё нужное на одной странице, без перегруза кнопками и рекламой*

### Проблема, которую решает панель

Существующие решения делятся на два лагеря:  
- тяжёлые корпоративные панели (cPanel, Plesk, ISPmanager) → жрут 1–3+ ГБ RAM даже в простое, сложный интерфейс, много ненужного  
- «лёгкие» open-source панели → часто либо устаревшие, либо всё равно содержат кучу лишнего (почта, DNS, биллинг, многосерверность), либо ломаются после обновлений системы  

AdelPanel — это **осознанно урезанный** инструмент: только то, что реально нужно 90% веб-разработчиков и владельцев сайтов.

### Основные возможности

- **Одна команда установки** — весь стек (Nginx + PHP-FPM + MySQL/MariaDB + Let's Encrypt) разворачивается за 2–3 минуты  
- **Сайты / домены** — добавил домен → сразу готов Nginx-конфиг + бесплатный SSL-сертификат  
- **Файловый менеджер в браузере**  
  • drag-and-drop загрузка  
  • распаковка .zip / .tar.gz одним кликом  
  • встроенный редактор кода (без необходимости в FTP-клиенте)  
- **Базы данных** — создание базы + пользователя MySQL/MariaDB в один клик  
  • импорт/экспорт дампов прямо через панель  
- **Поддержка актуальных версий** PHP 8.1–8.3  
- **Минималистичный интерфейс** — всего три основных раздела:  
  • Сайты  
  • Базы данных  
  • Настройки сервера / системные инструменты  
- **Очень низкое потребление ресурсов** — комфортно работает даже на VPS с 1–2 ГБ RAM  

### Кому подойдёт

- Разработчикам (тестовые стенды, pet-проекты, лендинги, небольшие сайты)  
- Фрилансерам и небольшим веб-студиям  
- Тем, кто устал от aaPanel / HestiaCP / CyberPanel / Webmin на слабых серверах  
- Людям, которым НЕ нужны: почта на сервере, встроенный DNS, биллинг, мониторинг трафика, многосерверное управление и т.п.

### Требования к серверу

- **ОС**: Ubuntu 20.04 / 22.04 / 24.04 или Debian 11 / 12  
- Root-доступ  
- Чистая установка (лучше ставить на свежий сервер без другого софта)
