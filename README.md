# AdelPanel Landing Page

Это исходные файлы **официального сайта** проекта AdelPanel — https://adelpanel.net

Ветка `landing` содержит статический landing page, который развёрнут через GitHub Pages на домене adelpanel.net.

### Что здесь находится

| Файл              | Описание                                      |
|-------------------|-----------------------------------------------|
| `index.html`      | Главная страница сайта — визитка панели       |
| `demo.html`       | Страница демонстрации (скриншоты / пример интерфейса) |
| `panel.jpg`       | Основной скриншот дашборда AdelPanel          |
| `adelpanel.ico`   | Фавикон сайта и проекта                       |
| `CNAME`           | Настройка кастомного домена adelpanel.net     |
| (возможно) стили и скрипты | CSS / JS файлы оформления (если добавлены)    |

### Основная идея страницы

Показать за 10–20 секунд главные преимущества AdelPanel:

- Очень простая и быстрая установка  
- Минимальное потребление ресурсов  
- Только нужные функции: сайты, файлы, базы данных  
- Автоматический Nginx + Let's Encrypt SSL  
- Без лишнего: почты, DNS, биллинга, многосерверности

Страница сделана максимально лаконично, на русском языке, с одной большой командой установки:
